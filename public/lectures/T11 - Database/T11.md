# SQL commands

Create table

```sql
CREATE TABLE IF NOT EXISTS
  todos (
    id serial PRIMARY KEY,
    title VARCHAR(50) NOT NULL,
    created_at TIMESTAMPTZ NOT NULL DEFAULT NOW()
  );
```

Delete table

```sql
DROP TABLE IF EXISTS todos;
```

---

View data

```sql
SELECT * from todos;
```

Create data

```sql
INSERT INTO
  todos (title)
VALUES
  ('New Todos');
```

Delete row

```sql
DELETE FROM
  todos
WHERE
  id = 1;
```
